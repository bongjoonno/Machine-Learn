# Machine Learn
A Machine Learning Library for Prediction and Categorization

## Features:

### Models
- Linear Regression
- Logistic Regression
- Naive Bayes
- Learning Rate Optimizer w/Genetic Algorithm
- Genetic Algorithm Parameter Optimizer

### Data Wrangiling:
- Data Scaling
- Train-Test Split
- Train-Test-Validate Split


### Metrics:
- R^2
- Categorical Accuracy
